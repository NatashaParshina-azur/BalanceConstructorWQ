Подключение Google Sheets

Введи данные своей таблицы, чтобы конструктор автоматически подтягивал актуальные данные при нажатии «Обновить».

1Открой свою таблицу в Google Sheets
2Файл → Поделиться → Настройки доступа → «Все у кого есть ссылка»
3Скопируй ID из адресной строки:
docs.google.com/spreadsheets/d/ЭТОТ_ID_СЮДА/edit

Если таблица приватная — нужен API Key от Google Cloud Console (Sheets API v4). Для публичных таблиц не нужен.
